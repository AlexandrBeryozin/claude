# Track Record — точность рекомендаций CS2 Signal Bot

Формат записи: `дата рекомендации | предмет | цена тогда | цена сейчас | % изменения | статус`

Проверка проводится в 00:00 UTC для рекомендаций возрастом 7-14 дней
(price-history.json / Price Overview API).

## Журнал проверок

_2026-07-01: инициализация файла. Это первый запуск системы — предыдущих
рекомендаций для проверки нет. Текущее время запуска не совпадает с 00 UTC,
поэтому плановая проверка в этом запуске не проводилась. Первая реальная
проверка состоится не раньше 2026-07-08 (когда рекомендациям из этого
запуска исполнится 7 дней) и должна запускаться в окне 00:00 UTC._
